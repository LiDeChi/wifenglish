## Dialogues / Role-Play Prompts

### Dialogue 1 (Prompt)
```markdown
Customer: Can you order now?
(你可以下订单了吗？)
```

### Dialogue 2 (Prompt)
```markdown
Customer: Yes, let's make the order now.
(可以，现在就下订单吧。)
```

### Dialogue 3 (Dialogue)
```markdown
Seller: When can you place the order?
(您什么时候能下单？)
Customer: I will come again tomorrow.
(我明天再来。)
```

### Dialogue 4 (Dialogue)
```markdown
Seller: If you order today, I can give you 10% discount.
(如果您今天下单，我可以给您九折优惠。)
Customer: OK, will we place the order now?
(好，那我们现在下订单吧？)
```
