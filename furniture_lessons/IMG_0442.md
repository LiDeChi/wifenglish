## Dialogues / Role-Play Prompts

### Dialogue 1 (Asking for dimensions)

```markdown
Customer: Can you tell me the size of it?
Seller: Yes, the length is 1.8 m, the width is 0.6 m and the height is 80 cm.
(Example follow-up): Or, we have a three-seater sofa. It has a length of 2.4 m, the width is 0.9 m and the height is 80 cm.
```

### Dialogue 2 (Asking for alternative sizes)

```markdown
Customer: Do you have any other size?
Seller: Yes, we have a king-size one. It has a length of 2.4 m.
(Example follow-up): Or, we have a three-seater sofa. It has a length of 2.4 m, the width is 0.9 m and the height is 80 cm.
```

### Dialogue 3 (Asking for width)

```markdown
Customer: What's the width?
Seller: 1.5 m.
(Example follow-up): Or, we have a three-seater sofa. It has a length of 2.4 m, the width is 0.9 m and the height is 80 cm.
```

### Dialogue 4 (Asking for length)

```markdown
Customer: What's the length?
Seller: 1.2 m.
(Example follow-up): Or, we have a three-seater sofa. It has a length of 2.4 m, the width is 0.9 m and the height is 80 cm.
```
