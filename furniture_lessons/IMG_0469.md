## 💬 Dialogues / Role-Play Scenarios

### Dialogue 5: Delivery Time Inquiry

```markdown
Customer: If you order today, we can deliver them in 2 days.
如果今天下单，我们可以在两天内交货。

Seller: It's quite far away.
离得太远了。

Customer: If you can make sure that I can order now.
如果你能保证我现在下单的话。

Seller: Sure, we can deliver them in 2 days.
当然可以，我们可以在两天内交货。
```

### Dialogue 6: Payment and Shipping Options

```markdown
Customer: Is it free to send to Guangzhou?
发到广州是否免费？

Seller: Sorry, you need to pay 300 Yuan for the trucking. Only in Lecong, it's free delivery.
抱歉，你需要支付300元运费。只有在乐工，它是免费送货。
```
