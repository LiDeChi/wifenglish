# Price Negotiation (价格谈判)

***

### 📚 Vocabulary / Grammar Focus: Price Negotiation Phrases

(These phrases are used when discussing and negotiating prices.)

| English Phrase | Chinese Translation | Meaning/Usage |
| :--- | :--- | :--- |
| **low price** [lōu jià] | 低价的 | (Adjective phrase) |
| **best price** [zuì jià de] | 最好的价格 | (Superlative) |
| **reduce** [jiǎn yú] | 减少/降低 | (Verb) |
| **add** [jiā] | 增加 | (Verb) |
| **discount** [diǎn xiǎo] | 折扣 | (Noun) |
| **wholesale price** [gòuwù jià] | 批发价 | (Noun phrase) |

**Dialogue Flow:**

| Speaker | English Dialogue | Chinese Translation | Notes / Context |
| :--- | :--- | :--- | :--- |
| **Customer (顾客)** | I'm looking for a sofa. | 我在找一张沙发。 | Starting the inquiry. |
| **Salesperson (销售员)** | We have many styles. What kind of style do you prefer? | 我们有很多款式。您喜欢什么风格？ | Offering assistance. |
| **Customer** | Something modern and comfortable. How much is this sofa? | 现代又舒适的。这个沙发多少钱？ | Asking for the price. |
| **Salesperson** | This one costs $1,500. | 这个是$1,500。 | Stating the initial price. |
| **Customer** | Oh, that's a bit expensive. Can you give me a discount? | 哦，有点贵。能给我打折吗？ | Negotiating (asking for a discount). |
| **Salesperson** | We can reduce it to $1,300 if you buy two. | 如果您买两个，我们可以降到$1,300。 | Offering a deal/discount based on quantity. |
| **Customer** | Hmm... If I buy three, what price will you give me? | 嗯...如果我买三个，能给我什么价钱？ | Further negotiation (increasing quantity). |
| **Salesperson** | For three, we can give you 20% discount. | 三个的话，我们可以给您20%的折扣。 | Stating a percentage discount. |
| **Customer** | That sounds good. And do you offer any wholesale price? | 听起来不错。你们有批发价吗？ | Asking for the best possible deal (wholesale). |
| **Salesperson** | Yes, if you buy more than five, we can give you a special price. | 有的，如果您买超过五个，我们可以给您一个特别价格。 | Setting conditions for a better price. |
