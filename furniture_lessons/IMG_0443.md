# 🛋️ Furniture Showroom Dialogue Practice (家具店购物中心)

This page contains role-play dialogues simulating a shopping experience in a furniture store, presented bilingually (English and Chinese).

***

## Dialogues

### **Dialogue 6**
**(Focus: Dimensions - Height)**

| Speaker | English Text | Chinese Translation |
| :--- | :--- | :--- |
| **Customer:** | What's the height? | 多高？ |
| **Seller:** | 1.2m. | 一米二。 |

### **Dialogue 7**
**(Focus: Dimensions - Length)**

| Speaker | English Text | Chinese Translation |
| :--- | :--- | :--- |
| **Customer:** | Can you make it longer? | 可以把它弄长点吗？ |
| **Seller:** | Yes, how long do you want? | 是的，您想要多长？ |
| **Customer:** | 1.9m. | 一米九。 |

### **Dialogue 8**
**(Focus: Dimensions - Size/Capacity)**

| Speaker | English Text | Chinese Translation |
| :--- | :--- | :--- |
| **Customer:** | Do you have a bigger size? | 有没有大一点的尺寸？ |
| **Seller:** | Yes, I do. | 是的，有。 |
| **Customer:** | This sofa is too small. Do you have a bigger size? | 这个沙发太小了。有没有大一点的尺寸？ |
| **Seller:** | Yes, we have three-seater sofas. | 是的，我们有三座沙发的。 |
