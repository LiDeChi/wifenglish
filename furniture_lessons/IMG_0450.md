### 💬 Dialogue Role-Play / 对话情景模拟

*(This section simulates a conversation in a furniture showroom.)*

**Seller:** Good morning, welcome to our shop.
（早上好，欢迎光临我们的店。）

**Customer:** Good morning. Do you have any bed like this picture?
（早上好。你们有像这张图片上的床吗？）

**Seller:** Yes, sorry. Do you like any bed like this picture?
（是的，抱歉。您喜欢像这张图片上的哪种床？）

**Customer:** Oh, I want to buy a sofa.
（哦，我想买一个沙发。）

**Seller:** Sure. Would you like to have some others with similar?
（当然可以。您想要一些相似款式的吗？）

**Customer:** No, thank you. This model is sold out. But we have some others with similar.
（不，谢谢。这个款式卖完了。但是我们有其他类似的。）
